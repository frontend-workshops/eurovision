# HTML Semantico

## Problema: Il codice HTML non è semanticamente corretto, il che può causare problemi di accessibilità e SEO.

Il codice HTML fornito utilizza tag `<div>` per strutturare tutto il contenuto, inclusi titoli, paragrafi e informazioni di contatto. Questo approccio non è semantico e rende difficile per i motori di ricerca e gli screen reader comprendere la struttura e il significato del contenuto.

## Soluzione: Utilizzare tag HTML semantici per migliorare la struttura e la comprensione del contenuto.

Nell'attuale codice ci sono informazioni importanti come il **titolo dell'evento**, la **data**, la **descrizione** e i **dettagli di contatto**.
Come sviluppatore, rendi il codice HTML più semantico utilizzando tag appropriati.
